# /fix-me

このコマンドは、ソースコード内の特定のコメントを検索し、その指示に従って自動的に修正を行うためのコマンドです。

## 動作フロー

1.  **検索**: プロジェクト全体のソースコードから `// FIX-ME-CLAUDE: ` という文字列を検索します。
2.  **解析**: 見つかったコメントの `:` 以降に記述されている指示内容を確認します。
3.  **実装**: 指示内容に従って、該当箇所のコードを修正または実装します。
4.  **クリーンアップ**: 修正が完了したら、対象の `// FIX-ME-CLAUDE: ` コメントを削除します。

## 守るべきルール

- **規約の遵守**: 修正を行う際は、`.claude/rules` 配下にあるすべてのコーディング規約（`custom-components.md`, `style.md` など）に従うこと。
- **最小限の変更**: 指示された内容に対して、必要かつ最小限の変更を行うこと。
- **確認**: 指示内容が曖昧な場合や、影響範囲が広大になることが予想される場合は、独断で進めずユーザーに確認を行うこと。

## 例
**修正前**:
```tsx
// FIX-ME-CLAUDE: BBAppInputを使用して、name="email" のバリデーションを追加して
<input type="text" />
```

**修正後**:
```tsx
<BBAppInput
  name="email"
  register={register}
  error={errors.email}
/>
```
