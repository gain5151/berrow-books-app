# Coding Style & Conventions

- **Language**: TypeScript for all web code.
- **Components**: Use React Functional Components with Tailwind CSS.
- **Naming**: 
  - Components: PascalCase
  - Functions/Variables: camelCase
  - Files: kebab-case (e.g., `user-profile.tsx`)
- **State Management**: React Hooks (useState, useMemo, etc.).
- **Data Fetching**: Use Server Components where possible, otherwise standard fetch/API routes.
